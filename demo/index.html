Use PlayerVars

透過 playerVars 參數，您可以自訂 player 初始化時期的一些行為或介面，比如說自動播放、循環播放、條件式顯示瀏覽人數等。以限定瀏覽人數為例，你可以傳入 { playerVars: { showViewsCount: 300000 } } 來達到「當瀏覽人數達30萬時，才顯示瀏覽人數」的功能。
